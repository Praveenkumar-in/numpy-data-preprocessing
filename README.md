# 📊 NumPy Arrays & Data Normalization (Google Colab)

This project demonstrates **NumPy array operations** and **data preprocessing techniques** such as **Min-Max Normalization, Z-Score Standardization, and Frequency Distribution** using Python in **Google Colab**.

## 🚀 Features
- Convert Python list to **NumPy array**
- Check **array properties**
- **Reshape arrays**
- Create **multi-dimensional arrays**
- **Min-Max Normalization**
- **Z-Score Standardization**
- **Frequency Distribution using Sturges Rule**

## 🛠 Libraries Used
- NumPy
- Pandas
- Scikit-learn

## ▶️ Run in Google Colab
1. Open **Google Colab**
2. Upload the Python code or paste it into a notebook
3. Run the cells to see the output
